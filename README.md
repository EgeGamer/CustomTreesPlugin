# CustomTrees Plugin

## Beschreibung
Das CustomTrees Plugin ermöglicht es Benutzern, benutzerdefinierte Baumstrukturen in Minecraft zu erstellen, die in einer Vielzahl von Biomen vorkommen können.

## Installation
1. **Download:** Stellen Sie sicher, dass Sie die neueste Version des Plugins im GitHub-Repository herunterladen.
2. **Installation:** 
   - Legen Sie die .jar-Datei im `plugins`-Ordner Ihres Minecraft-Servers ab.
   - Starten Sie den Server neu, um das Plugin zu aktivieren.
3. **Abhängigkeiten:** Stellen Sie sicher, dass die erforderlichen Abhängigkeiten installiert sind, wie z.B. [WorldEdit](https://enginehub.org/worldedit/).

## Biome-Liste
Das CustomTrees Plugin unterstützt die folgenden Biome:
- Wald
- Regenwald
- Taiga
- Wüste
- Sümpfe
- Gebirge

## Litmatica Export Anleitung
Um Bäume mit dem Litmatica-Plugin zu exportieren:
1. Erstellen Sie die Baumstruktur in Ihrem Minecraft-Server.
2. Verwenden Sie den Befehl `/schematic save <name>` um die Struktur zu speichern.
3. Teilen Sie die gespeicherte Schematische Datei mit anderen Benutzern des Plugins.

## Leistungseinstellungen
Um die Leistung des Plugins zu optimieren:
- Verringern Sie die Baumkomplexität in den Einstellungen.
- Deaktivieren Sie animierte Effekte, die die Leistung beeinträchtigen könnten.
- Stellen Sie die Renderdistanz Ihres Servers auf einen optimalen Wert ein (z.B. 10 Chunks).

## Fehlersuche
- **Problem:** Das Plugin startet nicht.
  - **Lösung:** Überprüfen Sie, ob alle Abhängigkeiten installiert sind und ob die .jar-Datei im richtigen Ordner ist.
  
- **Problem:** Bäume werden nicht gerendert.
  - **Lösung:** Stellen Sie sicher, dass die Biome, in denen Sie die Bäume platzieren möchten, unterstützt werden.

- **Problem:** Leistungsprobleme.
  - **Lösung:** Überprüfen Sie die Leistungseinstellungen und optimieren Sie die Serverkonfiguration.
